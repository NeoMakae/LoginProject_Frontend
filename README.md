## Requirements

- Docker & Docker Compose installed on your machine  
- Optional: Node.js & Java if you want to run frontend/backend outside Docker

---

## Steps to Run the Application


1. Prepare a folder for Docker Compose (docker-compose.yaml)
2. Create a folder on your desktop (or any location):
3. move the docker-compose.yml from your frontend repo into this folder.
4. clone the backend and frontend repos to 2 separate folders 
5. run docker-compose up --build  on the folder where the 
6. docker-compose down
